### 简短变量声明


请记住“:=”是一个变量声明语句，而“=”是一个变量赋值操作。也不要混淆多个变量的声明和元
组的多重赋值，后者是将右边各个的表达式值赋值给左边对应位置的各个变量：

```
i, j = j, i // 交换 i 和 j 的值
```
